## Castra Prime Assets

This mod is purely a split of the graphical/sound assets from the Castra mod to support the new version, "Castra Prime". **None of these assets were created by me.**

Original mod: https://github.com/Bartz24/castra
https://mods.factorio.com/mod/castra

All assets created by original mod author [**Bartz24**](https://mods.factorio.com/user/Bartz24), except:
- Atom forge and fusion reactor assets by [Hurricane](https://mods.factorio.com/user/Hurricane046), some modifications by Bartz24
- Google Material Design used for enemy research shortcut icon - Apache License ver 2.0
- Gencris for higher definition technology icons and higher quality item icons
